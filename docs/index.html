Replication Steps and Goal

Please see the tabs above for the steps for replicating the data construction and network analysis for Austin, TX from The Civic Elite. The final analysis file can be also used to replicate results for Cleveland, OH if the data files are exchanged for the Cleveland data files here.

This repo is also meant to be a tutorial for constructing similar results for any city in the United States. For the time period 1998-2003, this can be done immediately with the code and data provided so long as the code is properly modified. In order to apply these analyses to other cities from 2010 onward, though, you need to use IRSx to construct a list of active filers in the the state and time period that you are interested in. IRSx provides helpful documentation for doing this. Once you have the active filer records, you can then match this data to the AWS board records provided on Dropbox.
